AUTOANKI PORTABLE FOR macOS
===========================

START
-----
Open AutoAnki.app. This build is unsigned, so the first launch may be blocked
by macOS. Control-click AutoAnki.app, choose Open, then choose Open again.

FIRST USE
---------
Open Settings inside AutoAnki, choose a provider, and enter your own API key.
The key is saved locally in the hidden .env file in this folder.

YOUR FILES
----------
data/       Your card stacks and custom prompt
exports/    Anki packages you create

Keep the complete AutoAnki folder together. Back up the data folder to preserve
your cards. No installation or administrator access is required.

This Apple Silicon build is intended for M1, M2, M3, M4, and newer Macs.
