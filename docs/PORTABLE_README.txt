AUTOANKI PORTABLE
=================

START
-----
Double-click AutoAnki.exe. The app opens in its own desktop window.

FIRST USE
---------
Open Settings inside AutoAnki, choose a provider, and enter your own API key.
The key is saved locally in the hidden .env file in this folder.

YOUR FILES
----------
data\       Your card stacks and custom prompt
exports\    Anki packages you create

Keep the complete AutoAnki folder together. You can move or copy the folder to
another Windows computer. Back up the data folder to preserve your cards.

No installation or administrator access is required. An internet connection is
only needed when generating cards through your selected AI provider.
